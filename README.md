# Swimming Pool Booking - Frontend

Vue 3 frontend for swimming pool booking system.

## Tech Stack
- Vue 3 (Composition API)
- Vue Router
- Tailwind CSS
- Vite

## Setup

```bash
npm install
npm run dev
```


## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
